# Elora Taurus
The Elora Taurus Project is an independent AI governance control-plane and runtime architecture initiative built around a security-first approach. It explores how AI systems can be structured when inference is treated as proposal and authorization is enforced at a deterministic commit boundary, with governance and replay implemented as explicit system functions rather than afterthoughts.

The system is developed and operated on self-managed infrastructure using open-source technologies, with clear execution boundaries, scoped authority, and inspectable policy evaluation. Observability and governance are separated by design: execution explains what happened, while the control plane determines whether the outcome was admissible and why.

Elora Taurus is not a product or commercial offering. It is an ongoing engineering project focused on applying infrastructure and security discipline to AI runtime design while contributing practical implementation insight toward an open and interoperable AI governance architecture. The goal is to explore systems that prioritise reproducibility, control, and operational clarity over abstraction or convenience.

The project is developed in public through architecture documentation, technical disclosure, and experimental implementation.
